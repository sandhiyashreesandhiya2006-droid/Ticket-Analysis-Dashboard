# Ticket Analysis Dashboard

## 📌 Overview
An interactive Power BI dashboard designed to analyze support tickets, monitor performance, and provide actionable insights through dynamic visualizations.

## ✨ Features
- Interactive dashboard
- Ticket status analysis
- Priority-wise insights
- Resolution time tracking
- KPI cards
- Dynamic filters and slicers

## 🛠 Technologies
- Power BI
- Power Query
- DAX
- Excel
