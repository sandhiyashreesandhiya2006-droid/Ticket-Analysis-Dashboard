# 🎫 Ticket Analysis Dashboard (Excel Automation)

An interactive Excel Automation Dashboard designed to analyze customer support tickets, monitor SLA performance, track ticket resolution, and generate actionable business insights.

## 🚀 Features

- Automated Excel data processing
- Interactive KPI Dashboard
- Ticket Status Analysis
- Priority Analysis
- Resolution Time Tracking
- SLA Performance Monitoring
- Agent Performance Analysis
- Trend Analysis
- Dynamic Charts & Visualizations

## 🛠 Tech Stack

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Power Query
- Excel Automation
- Data Visualization

## 📊 Dashboard Insights

- Total Tickets
- Open vs Closed Tickets
- Priority Distribution
- SLA Compliance
- Average Resolution Time
- Monthly Ticket Trends
- Agent Performance
